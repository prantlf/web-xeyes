# &lt;x-eyes&gt;

Shows a pair of eyes following movements of the mouse cursor on the page.

## Default

Default look &amp; feel:

<Preview withToolbar>
  <Story name="Default" parameters={{notes}}>
    {`
<x-eyes></x-eyes>
`}
  </Story>
</Preview>

## Delayed

Movement delayed by 200ms:

<Preview withToolbar>
  <Story name="Delayed" parameters={{notes}}>
    {`
<x-eyes delay=200></x-eyes>
`}
  </Story>
</Preview>

## Distance

Iris distance 5px:

<Preview withToolbar>
  <Story name="Distance" parameters={{notes}}>
    {`
<x-eyes distance=5></x-eyes>
`}
  </Story>
